省政府办公厅政府信息公开办公室：

为进一步加强人社厅网站的建设、管理，根据《国务院办公厅关于印发政府网站发展指引的通知》（国办发〔2017〕47号）和《河南省人民政府办公厅关于2018年第二季度全省政府网站建设管理情况的通报》（豫政办文〔2018〕27号）要求，我们对人社厅所属网站的信息更新情况、空白栏目、服务事项公开、网民互动、网页可用性、网站安全等问题进行了抽查检查，抽查的河南省人力资源和社会保障厅、河南省公务员局和中国中原人才网三个网站全部合格。

在此次政府网站季度抽查检查中，针对网站出现的死链接、信息更新不及时等问题，我们十分重视，进行了相应的通报及整改，要求保证网站链接的真实性、可用性；保证信息公开的时效性和准确性；同时针对网站的政务咨询、调查征集等互动类栏目要及时回应，确保及时了解民意、解答群众疑问并接受群众的意见建议。

下一步，为加强网站的日常监管和内容保障，我们将进一步对网站的信息更新、信息内容质量、网站内容建设和互动回应情况等进行日常监测、抽查及整改；同时针对网站的安全稳定性，我们将加大保障力度，进行定期维护，做好风险防范和漏洞处理，确保网站安全运行，保持信息渠道畅通，为群众提供良好的网站办事和查询体验，不断增强群众满意度。
